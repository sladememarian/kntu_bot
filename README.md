## рҹҡҖ Setup

### 1. Install Python 3.10+

### 2. Clone & Install Dependencies
```bash
cd telegram_bot
pip install -r requirements.txt
```

### 3. Configure Environment Variables
| Variable | Required | Description |
|----------|----------|-------------|
| `BOT_TOKEN` | вң… | From [@BotFather](https://t.me/BotFather) |
| `GOOGLE_API_KEY` | вң… | Google AI Studio key (for `/ai`, `/imagine`) |
| `DATABASE_URL` | в¬ң | PostgreSQL URL (falls back to JSON) |
| `ADMIN_IDS` | в¬ң | Comma-separated Telegram user IDs |

### 4. Run the Bot
```bash
python app.py
```

### 5. Add to Group
1. Open [@BotFather](https://t.me/BotFather) вҶ’ `/mybots` вҶ’ select your bot
2. **Bot Settings** вҶ’ **Allow Groups** вҶ’ Turn ON
3. **Group Privacy** вҶ’ Turn OFF (so bot can track members)
4. Add the bot to your group and make it admin
